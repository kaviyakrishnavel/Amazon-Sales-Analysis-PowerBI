# Amazon Sales Analysis-Power BI
## Project Overview
This project analyzes Amazon sales data using Power BI to identify sales trends,regional performance,and product-wise insights.

## Tools Used
- Power BI
- Excel (for initial data cleaning)

## Key Insights
- Total Sales and Profit overview
- Sales by Region
- Sales by Product Category
- Date-wise sales trend analysis

## Features
- Interactive dashboard
- Slicers for Order Date and Category
- Visuals inculde KPI cards,bar charts, and slicers
